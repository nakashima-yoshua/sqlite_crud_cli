# SQLite CRUD CLI (Rust)

SQLiteを使ったシンプルなメモ管理CLIアプリです。Rustで構築されたこのCLIツールを使えば、以下のような基本的な操作が簡単にできます。

- メモの追加
- メモの一覧表示
- メモの更新
- メモの削除

---

## 🔧 使用技術

- Rust
- SQLite
- [rusqlite](https://docs.rs/rusqlite)（SQLiteバインディング）
- [clap](https://docs.rs/clap/latest/clap/)（CLI引数パーサー）
